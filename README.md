# Hướng dẫn Cài đặt & Chạy Dự án Thu - Chi

Hiện tại máy tính của bạn chưa có **Node.js**, là môi trường cần thiết để chạy các ứng dụng React hiện đại.

## Bước 1: Cài đặt Node.js
1. Tải bộ cài đặt Node.js (phiên bản LTS) tại: [https://nodejs.org/](https://nodejs.org/)
2. Chạy file cài đặt và làm theo hướng dẫn (Next -> Next -> Install).
3. Sau khi cài xong, **Khởi động lại máy tính** hoặc tắt/bật lại phần mềm VS Code/Terminal để cập nhật đường dẫn.

## Bước 2: Cài đặt thư viện
Mở Terminal tại thư mục dự án (`d:\My vscode\thuchi`) và chạy lệnh:
```bash
npm install
```

## Bước 3: Chạy ứng dụng
Sau khi cài đặt xong, chạy lệnh:
```bash
npm run dev
```
Trình duyệt sẽ mở ứng dụng tại địa chỉ `http://localhost:5173`.
